Welcome to the CobraPin Pinball Controller documentation!
===================================

.. note::

   This project is under active development.

**CobraPin** is a pinball controller designed to offer a basic all-in-one assembled affordable solution. It is intended to be hosted by a computer running the Mission Pinball Framework `MPF <https://missionpinball.org/>`_ and is based on the Open Pinball Project `OPP <https://pinballmakers.com/wiki/index.php?title=OPP>`_ It was released to the public in 2021 on Kickstarter. There is now an ecosystem of electronics and options for CobraPin.

Don't forget to subscribe to `my Youtube channel <https://www.youtube.com/@thomasfulenwider>`_ for more info and updates!
